Тег <form> використовується для створення HTML-форми для введення даних користувачем.

Елемент <select> використовується для створення розкривного списку. Елемент <select> найчастіше використовується у формі для збору введених даних користувача. Теги <option> всередині елемента <select> визначають доступні параметри у спадному списку. Доступні атрибути: autofocus, disabled, name, required, multiple. (застосування аналогічне з елементами input)

Тег <textarea> визначає багаторядковий елемент керування введенням тексту. Елемент <textarea> часто використовується у формі для збору введених користувачів, наприклад коментарів або оглядів. Розмір текстової області визначається атрибутами і (або за допомогою CSS). Доступні атрибути: autofocus, disabled, maxlength, name, placeholder, readonly, required. (застосування аналогічне з елементами input)
